# 🧩 Interview Patterns Playbook

Cross-cutting patterns that appear frequently in coding interviews.

## Patterns

- **Fast and Slow Pointers** — Cycle detection, finding middle elements
- **Merge Intervals** — Overlapping intervals, scheduling problems
- **Top-K Elements** — Heap-based solutions for finding top/bottom K
- **Subsets, Permutations, Combinations** — Backtracking templates
- **Islands and Connected Components** — Grid-based graph traversal
- **Tree BFS and DFS Patterns** — Level-order variations, path problems

Each pattern includes templates, visual diagrams, and curated problems.
