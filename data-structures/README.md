# 📦 Data Structures

Core data structures with implementations, complexity analysis, and curated problems.

## Categories

- **Linear:** Arrays, Strings, Linked Lists, Stacks, Queues
- **Hash-Based:** Hash Tables, Hash Maps, Hash Sets
- **Hierarchical:** Binary Trees, BST, AVL Trees, Tries
- **Graph-Based:** Graphs, Union-Find
- **Advanced:** Heaps, Segment Trees, Fenwick Trees, Monotonic Structures
- **Design:** LRU Cache, LFU Cache

Each data structure includes theory, implementation, complexity tables, and interview problems.
