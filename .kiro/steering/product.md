# Product Overview

## Project Identity

**DSA Mastery** is a world-class, interview-ready Data Structures & Algorithms repository targeting **FAANG and top-tier software engineering roles** (Google, Meta, Amazon, Apple, Microsoft, and competitive startups).

## Target Audience

- **Primary**: Software engineers preparing for technical interviews at top tech companies
- **Secondary**: Computer science students building strong DSA fundamentals
- **Tertiary**: Self-taught developers seeking structured, comprehensive learning resources

## Key Features

### Visual-First Approach
- **GitHub-native Mermaid diagrams** for every concept (no external images)
- Multiple diagram types: flowcharts, state machines, sequence diagrams, mindmaps
- Visual representations of data structure layouts and algorithm execution flows

### Interview-Focused Content
- **Aligned with Neetcode 150 and Blind 75** problem sets
- Pattern-based problem solving (Fast & Slow Pointers, Sliding Window, Top-K, etc.)
- Company-tagged problems organized by Google, Meta, Amazon, etc.
- Interview intuition sections with signal words and recognition patterns

### Comprehensive Documentation
- Every topic includes: definition, visual representation, complexity analysis, step-by-step walkthrough, edge cases, and curated problems
- Dual-purpose content: learning resource + quick-reference cheat sheet
- Complete coverage: foundations, data structures, algorithms, and interview patterns

### Clean, Production-Quality Code
- Type-hinted Python 3.12+ implementations
- Comprehensive docstrings with complexity analysis
- Built-in test cases for every implementation
- Stdlib-only (no external dependencies unless explicitly noted)

## Philosophy

**Clarity over cleverness.** Each implementation serves as both a working solution and a teaching tool. Content is written as if explaining to a smart peer who has never seen the concept before, while also being a quick-reference guide the night before an interview.

## Differentiation

Unlike typical DSA repositories:
- **Visual-first**: Every concept has multiple Mermaid diagrams
- **Interview-ready**: Pattern recognition and signal words for problem identification
- **Comprehensive**: No placeholders or TODOs—every section is complete
- **Structured**: Consistent templates and quality checklists ensure uniformity
- **Curated**: 5-10 must-solve problems per topic with difficulty and pattern tags
