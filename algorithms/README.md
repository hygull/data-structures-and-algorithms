# ⚙️ Algorithms

Fundamental algorithmic techniques and problem-solving paradigms.

## Categories

- **Sorting & Searching:** All major sorting algorithms, binary search variations
- **Techniques:** Two Pointers, Sliding Window, Divide & Conquer
- **Paradigms:** Greedy, Dynamic Programming, Backtracking
- **Graph Algorithms:** Shortest Path, MST, Topological Sort
- **String Algorithms:** Pattern matching (KMP, Rabin-Karp)

Each algorithm includes theory, implementation, complexity analysis, and curated problems.
