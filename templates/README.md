# 🛠️ Templates

Reusable templates for maintaining consistency across the repository.

## Available Templates

- **topic-readme-template.md** — Standard format for topic documentation
- **problem-readme-template.md** — Standard format for problem solutions
- **code-template.py** — Boilerplate for Python implementations

## Usage

All new topics and problems should follow these templates to ensure:
- Consistent structure and formatting
- Complete documentation
- Proper complexity analysis
- Comprehensive test cases

See CONTRIBUTING.md for detailed guidelines on using these templates.
